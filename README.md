📌 **Objetivo:** Determinar si un paciente será readmitido en menos de 30 días.

🛠 **Herramientas:** Python (pandas, scikit-learn, XGBoost)  
📊 **Técnicas:** Análisis exploratorio (EDA), modelos de clasificación (Random Forest, XGBoost).
